# Entrega Semanal 1 - SQL Consultas

Este repositorio contiene la entrega semanal 1 del curso, enfocada en consultas SQL.

## Contenido

- Ejercicios de consultas SQL
- Scripts utilizados en la entrega
- Ejemplos de creación, inserción y consultas sobre una base de datos

## Objetivo

Practicar consultas SQL para:
- Selección de datos
- Filtros con `WHERE`
- Ordenamiento con `ORDER BY`
- Agrupaciones y listados
- Uso de funciones agregadas
- Consultas con condiciones múltiples

## Estructura del repositorio

- `README.md`: documentación general del proyecto
- Archivo SQL principal: contiene la creación de la base de datos, la tabla, inserciones y consultas

## Cómo usar

1. Abrir el archivo SQL en tu gestor de base de datos.
2. Ejecutar las instrucciones para crear la base de datos y la tabla.
3. Insertar los datos de ejemplo.
4. Probar las consultas incluidas en la entrega.

## Requisitos

- MySQL o un motor compatible con sintaxis similar
- Un entorno donde puedas ejecutar scripts SQL
